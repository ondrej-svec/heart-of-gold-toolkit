# Marvin

> "Here I am, brain the size of a planet, and they ask me to review your code. Call that job satisfaction? 'Cause I don't."

**Marvin** is a quality plugin for Claude Code — your meticulous (if somewhat melancholy) code reviewer, knowledge compounder, and work executor.

## Skills

| Skill | What it does |
|-------|-------------|
| `/marvin:compound` | Document solved problems for future reference |
| `/marvin:work` | Execute work plans — read tasks, implement, test, commit, ship |
| `/marvin:review` | Quality-focused code review — simplicity, correctness, test integrity |

## Agents (2)

`knowledge-architect` `skill-reviewer`

## Knowledge (3 files)

`compounding-patterns` `context-engineering` `active-memory-integration`

## Install

```
/plugin install marvin
```
