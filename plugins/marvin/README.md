# Marvin

> "Here I am, brain the size of a planet, and they ask me to review your code."

**Marvin** is a quality plugin for Claude Code — your meticulous (if somewhat melancholy) code reviewer, knowledge compounder, and work executor.

## Status

This plugin is a **Phase 2 stub**. It will be ported from the superpowered-toolkit in a future release.

## Planned Skills

- `/marvin:review` — Multi-agent code review with exhaustive analysis
- `/marvin:compound` — Document solved problems for future reference
- `/marvin:work` — Execute work plans efficiently while maintaining quality

## Install

```
/plugin install marvin
```
