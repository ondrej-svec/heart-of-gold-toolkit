# Babel Fish

> "The Babel fish is small, yellow, leathery, and probably the oddest thing in the universe."

**Babel Fish** is a media generation plugin for Claude Code — your Universal Translator for creating audio, images, and video content from text.

## Status

This plugin is a **Phase 3 stub**. It will be built in a future release.

## Planned Skills

- `/babel-fish:audio` — Text-to-speech and podcast generation
- `/babel-fish:image` — AI image generation and editing
- `/babel-fish:video` — Video content creation

## Install

```
/plugin install babel-fish
```
