# Deep Thought

> "The Answer to the Great Question... Of Life, the Universe and Everything... Is... Forty-two."

**Deep Thought** is a reasoning plugin for Claude Code — your personal Answer Computer for brainstorming, planning, investigating, and deep thinking.

## Status

This plugin is a **Phase 2 stub**. It will be ported from the superpowered-toolkit in a future release.

## Planned Skills

- `/deep-thought:brainstorm` — Explore ideas through collaborative dialogue
- `/deep-thought:plan` — Transform feature descriptions into structured project plans
- `/deep-thought:think` — Deep reasoning for complex decisions
- `/deep-thought:investigate` — Follow evidence trails to find root causes

## Install

```
/plugin install deep-thought
```
