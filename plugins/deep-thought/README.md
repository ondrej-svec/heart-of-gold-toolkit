# Deep Thought

> "The Answer to the Great Question... Of Life, the Universe and Everything... Is... Forty-two."

**Deep Thought** is a reasoning plugin for Claude Code — your personal Answer Computer for brainstorming, planning, investigating, and deep thinking.

## Skills

| Skill | What it does |
|-------|-------------|
| `/deep-thought:brainstorm` | Explore ideas through collaborative dialogue |
| `/deep-thought:plan` | Transform feature descriptions into structured project plans |
| `/deep-thought:think` | Deep reasoning — expert panels, devil's advocate, tradeoff analysis |
| `/deep-thought:investigate` | Follow evidence trails to find root causes (Sherlock/Poirot/Columbo) |
| `/deep-thought:review` | Multi-agent code review — security, performance, architecture, strategic |
| `/deep-thought:architecture-review` | Deep architectural review — failure modes, scaling, ADRs |
| `/deep-thought:cto` | Strategic CTO advisor grounded in the leadership canon |
| `/deep-thought:craft-skill` | Meta-skill that generates and refines SKILL.md files |
